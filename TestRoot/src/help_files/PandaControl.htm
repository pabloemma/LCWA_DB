Help File for the LCWA control gui























After starting the gui, you first need to define the input file. Usually, this will the original csv file , created by Rob.







Then choose the outputfile, which will be a csv file with a reduced number of entries. The program has a default list of variables which will be in the reduced file:

ReduceList = ["apMac","cpuUsage","deviceIp","deviceName","dtCreate","lanRxBytes", \

"lanRxErrors","lanRxPackets","lanTxBytes","lanTxErrors","lanTxPackets",\

"loadavg","memBuffers","memFree","memTotal","remoteIP",\

"remoteMac","wlanRxBytes","wlanRxErrors","wlanRxErrOther",\

"wlanRxErrRetries","wlanRxPackets","wlanRxRate","wlanTxBytes",\

"wlanTxErrors","wlanTxPackets","wlanTxRate"]



If you want to modify this list to look at other vraiables, add to them or remove, go to the menu Services and choose Get Variables. This will bring up all the variables which were in the original file. The default ones are already checked, you can add variables by checking and unchecking boxes.
